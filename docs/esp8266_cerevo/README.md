ESP8266 CEREVO 
===============
ESP8266 CEREVO  - Arduino Shield.

This Shield has ESP-WROOM-02, LED and button.
[ESP-WROOM-02](http://espressif.com/en/products/wroom/) is WiFi Module using ESP8266 chipset
[CEREVO](http://cerevo.shop-pro.jp/?pid=91592223) sales this WiFi modue and breakout board.
